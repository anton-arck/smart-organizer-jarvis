# 🤖 J.A.R.V.I.S. | Smart Downloads Agent
> **"Protocolo de orden activado, Señor."**

Un gestor de archivos autónomo desarrollado con **Python 3.14** y **Flet**, diseñado para mantener tu carpeta de descargas impecable mediante el uso de observadores de eventos del sistema de archivos.

## 🚀 Stack Tecnológico
* **Frontend:** [Flet](https://flet.dev/) (Material Design 3)
* **Backend:** Python 3.14 + OOP
* **Monitor:** [Watchdog 6.0](https://pypi.org/project/watchdog/) (Event-driven architecture)
* **Vigilancia:** Multi-threading para procesos de fondo.

🧠 Arquitectura del Sistema

El sistema se divide en tres capas principales:

    Core Organizer: Lógica de clasificación por extensiones.

    Sentinel Monitor: Observador asíncrono que detecta nuevos archivos.

    Cyber-HUD: Interfaz de usuario que reporta acciones en tiempo real.
