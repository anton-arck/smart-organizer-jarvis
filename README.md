# J.A.R.V.I.S. | Smart File Organizer 🤖

Asistente virtual desarrollado en **Python** con **Flet** para la gestión automática de archivos. 

## ✨ Características
- **Modo Centinela:** Vigilancia en tiempo real de la carpeta Downloads.
- **Categorización Inteligente:** Clasifica archivos por extensión (PDF, Imágenes, Código).
- **Interfaz Cyberpunk:** HUD futurista con reportes de misión en vivo.

## 🛠️ Instalación
1. Clonar el repo: `git clone ...`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar: `python main.py`
